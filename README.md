**MX Events** is a library for generic event handling in JavaScript.

## Prerequisites

You will need [Node.js](https://nodejs.org/) **version 8+**.

## Instalation

npm:

```
npm install @mx-js/events
```

yarn:

```
yarn add @mx-js/events
```
